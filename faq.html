Frequently Asked Questions

  • What is Etherparty?

    Etherparty is an easy-to-use smart contract creator that allows users to compile and deploy smart contracts without the knowledge of coding smart contract syntax.

  • Do you accept Bitcoin?

    Not at this time! Etherparty is powered by Fuel, which is an ERC20 token

  • What is an ERC20 token?

    The ERC20 token standard describes the functions and events that an Ethereum token contract has to implement.

  • Do I need Ethereum?

    Yes. You will need to Ethereum to purchase Fuel tokens, as well as pay the Ethereum transaction fees (gas).
